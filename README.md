# 🤖 **Deep Learning Mini Projects**
---

This repository contains a collection of mini-projects focused on **Deep Learning**, where different architectures and applications. Throughout the projects, different areas are explored, such as convolutional neural networks (CNNs) for computer vision tasks, natural language processing (NLP) for text analysis and understanding, and transformer-based models for more complex sequence and attention-related problems, among other approaches.

- 🖼️ **Convolutional Neural Networks (CNNs)** for computer vision tasks  
- 📚 **Natural Language Processing (NLP)** for text analysis and understanding  
- 🤖 **Transformers** for complex sequence problems and attention mechanisms  
- 🚀 **Transfer Learning** and fine-tuning of pre-trained models
